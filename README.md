# 🎉 SillyTavern Persona Collapse (人设折叠)

还在为 SillyTavern 左侧列表里长长的一长串“同一角色的不同换装/不同设定卡”感到头疼吗？找一张特定的卡要向下划半天？
隆重向大家介绍这把左侧栏的“收纳神器”—— **Persona Collapse (人设折叠)**！

这是专为**多马甲/多设定党**量身定制的拓展插件。它能将同一角色的各种变体卡片折叠成“主副卡分支”的树状结构，让你的界面瞬间清爽干净！

---

## ✨ 核心亮点功能

### 🗂️ 1. 丝滑的分支折叠体验
你可以自由指定一张卡片为“主卡”，将其他换装卡、差分卡作为分支收纳进去。
在左侧栏，主卡右上角会显示一个**精致低调的半透明毛玻璃角标**（提示里面收纳了几张卡）。点击主卡后，专属的“人设分支”面板会自动在下方展开，随时随地快速切换！

### 🤖 2. 懒人福音：双轨全自动收纳
不想手动一张张整理？插件支持全自动归类（可在顶部拓展设置面板开启）：
- **按同名前缀收纳**：自动将名字前缀相同的人设归入一组（如 \Alice_战斗服\ 会自动归入 \Alice\ 之下）。
- **按角色绑定收纳**：如果你在原版酒馆里为卡片绑定了特定角色，插件会自动读取绑定关系，把它们智能打包在一起！

### 🛠️ 3. 极速“批量管理”控制台
点击独立角色卡下方的 \[🛠️ 管理分支卡片]\ 按钮，即可呼出强大的双栏管理面板！
左侧是角色库，右侧是当前分支。你可以使用自带的**搜索框**和**快捷筛选器（同名/同绑定）**，一键选中多张卡片，瞬间完成海量分支的组建与拆解。

### ✏️ 4. 自定义分支标题
“人设分支”标题可以直接点击铅笔重命名，例如“爱丽丝套装”“战斗差分”“IF 线人格”。清空标题后会自动恢复默认名称。

### 📋 5. 快速复制人设
每张主卡和分支卡旁都有复制按钮。点击即可复制完整人设信息（不复制绑定关系），生成 `原名_副本`，并自动加入当前分支，适合快速制作换装、差分和实验版本。

### 🌍 6. 无缝原生融合
无论是全局的角色选择弹窗，还是右键菜单，插件都做了深度原生适配。
弹窗里只显示主卡，绝不让分支卡出来占位置；支持顺滑的拖拽排序与放入，拥有近乎原生自带功能般的完美体验！

## 🚀 安装与使用指引

📦 **通过 SillyTavern 安装**：
直接使用本仓库链接安装即可。`main` 分支是纯发布版，只包含运行插件所需文件：

```text
manifest.json
style.css
dist/index.js
README.md
LICENSE
```

安装后重启酒馆，在顶部拓展面板中确认已勾选“启用人设折叠功能”即可开始使用。

🧑‍💻 **开发源码**：
源码、构建配置和打包脚本已放在 `source` 分支，普通用户无需下载。

---

💡 **小贴士**：
如果你是第一次使用，建议先在左侧点选人设，然后点击下方的 \[🛠️ 管理分支卡片]\ 按钮，试试全新的搜索归纳功能，几秒钟就能把几十张卡整理得服服帖帖！
